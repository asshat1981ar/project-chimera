```kotlin
@Query("UPDATE crafting_recipes SET is_discovered = 1 WHERE required_npc = :npcId")
suspend fun discoverByNpc(npcId: String)
```

Add `updateChapterTag` to `SaveRepository.kt`:

```kotlin
suspend fun updateChapterTag(slotId: Long, tag: String) {
    val slot = saveSlotDao.getById(slotId) ?: return
    saveSlotDao.upsert(slot.copy(chapterTag = tag))
}
```
